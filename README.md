# Tissite: An IoT-Based Agricultural Renaissance 🌱

Tissite is a cutting-edge Smart Agriculture platform developed at **United International University (UIU)**. It integrates **Internet of Things (IoT)** surveillance with **Artificial Intelligence (AI)** to empower farmers with real-time field monitoring and instant plant disease diagnosis.

---

## 🚀 Key Features

- **AI-Driven Disease Prediction**: Upload leaf images to identify diseases (e.g., Common Rust) with high precision using image processing and AI.
- **Real-Time IoT Surveillance**: Live 24/7 video streaming from the field to ensure crop security and health.
- **Environmental Monitoring**: Track Soil Moisture, Temperature, and Humidity in real-time through an interactive dashboard.
- **Smart Irrigation Alerts**: Automated system notifications based on soil moisture levels to optimize water usage.
- **Responsive Dashboard**: A premium, mobile-friendly interface designed for ease of use by farmers and researchers.
